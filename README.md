A modern, web-based stock visualization tool built with FastAPI and Vue 3.

## Features
- **Interactive Charts**: Powered by TradingView Lightweight Charts.
- **Custom Indicators**: Pre-configured MA lines (17, 45, 117, 189, 305, 494 days).
- **Market Support**: Support for TW (Taiwan) and US markets with auto-color adjustment.
- **Responsive Design**: Mobile-friendly sidebar and layout.

## Tech Stack
- **Backend**: Python FastAPI, yfinance
- **Frontend**: Vue 3, Tailwind CSS, Lightweight Charts

## Quick Start
```bash
uv run main.py
```
Visit `http://localhost:8000`.
